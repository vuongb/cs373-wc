About Team UNICORN STEROIDS

The World Crises Database is a project for CS 373: Software Engineering from the University of Texas at Austin. The class is overseen by Professor Glenn Downing. The World Crises Database is a centralized location on the interwebs that gives users easy, direct access to information for past and present worldwide crises, as well as associated organizations and individuals.

The objective of the project is to implement a python webapp via Google App Engine that takes advantage of the cloud datastore.

Data is uploaded to the site via XML instances that conform to an XML schema, created and standardized by the class.

Various other requirements are given, including the creation of a UML diagram to represent the GAE datastore, a technical report, and unit testing.

The project was divided into three parts, which can be viewed here, here, and here. (Links available until 1/13)

The front-end work is done with HTML, CSS3, Twitter Bootstrap, and Django Templating.

In order to properly meet requirements and deadlines, the team is composed of six members:

Ryan Prater, Michael Tarng, Bryan Vuong, Boris Jonica, Dereck Sanchez, Cassie Schwendiman.

Collaboration and issue tracking was done via GitHub.
